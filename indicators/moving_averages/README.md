# Moving Averages

This folder contains implementations of various moving average indicators for technical analysis.

## Overview

Moving averages are fundamental technical analysis indicators used to identify trends and smooth price data by filtering out noise from random short-term price fluctuations.

## Contents

- Simple Moving Average (SMA)
- Exponential Moving Average (EMA)
- Weighted Moving Average (WMA)
- And more...

## Usage

Documentation and implementation details for each moving average type will be added to this folder.
